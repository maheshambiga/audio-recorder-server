## Core Libraries Used
    express@4.16.0
    google-cloud/storage@1.4.0
    jsonwebtoken@7.3.0

## Steps for setting up code

##### Setup
Install all the dependencies and dev-dependencies listed in package.json.

     $ npm install

##### To start the app in production mode:

    $ npm start
